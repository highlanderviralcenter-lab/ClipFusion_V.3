# Mapa do Sistema ClipFusion V3

Data de geração: $(date)

Esta árvore descreve a estrutura de diretórios e arquivos do ClipFusion V3.  Todos os
arquivos são gerados por este instalador e a fonte original é este próprio
arquivo final.

- **run/**: scripts para executar a aplicação, fila e preflight
- **installers/**: scripts de instalação e perfil
- **infra/**: camada de persistência (SQLite)
- **core/**: módulos de segmentação, decisão, renderização, transcrição
- **anti_copy_modules/**: fábrica de planos de proteção
- **viral_engine/**: heurísticas de arquétipos, ganchos e plataformas
- **gui/**: interface gráfica completa
- **app/**: CLI e worker de fila
- **config/**: arquivos de configuração YAML
- **docs/**: documentação
- **requirements/**: listas de dependências
- **output/**: relatórios gerados pelo sistema

Fonte original = próprio arquivo final
